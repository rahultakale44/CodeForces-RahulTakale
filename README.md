## ✅ Topics to Cover

### 1. Implementation

Direct simulation, condition handling, loops, input processing, and careful translation of problem statements into code.

### 2. Mathematics

Divisibility, factors, GCD, LCM, modular arithmetic, formulas, and mathematical observations.

### 3. Greedy

Choosing the best local decision and proving why it produces the correct final answer.

### 4. Number Theory

Prime numbers, factorization, divisors, modular operations, and number-based patterns.

### 5. Arrays & Strings

Traversal, frequency counting, transformations, comparisons, sorting, and pattern-based problems.

### 6. Prefix Sum

Efficient range calculations and cumulative frequency techniques.

### 7. Binary Search

Searching sorted data and applying binary search on the answer.

### 8. Two Pointers

Efficient traversal using two indices for arrays and strings.

### 9. Graphs

DFS, BFS, connected components, shortest paths, and graph traversal basics.

### 10. Dynamic Programming

State definition, transitions, memoization, tabulation, and optimization.
